# 👋 Hi, I'm Rohit Kumar

🔋 Embedded & EV Systems Engineer | BMS | CAN | System Integration  

Designing, developing, and validating EV systems with strong expertise in Battery Management Systems (BMS), battery pack architecture, and vehicle-level integration.

---

## 🚀 Expertise
- 🔋 Battery Pack Design & Development (2kW–5kW)  
  Cell selection, pack architecture, interconnect design, and system optimization  

- ⚙️ BMS Validation & Testing  
  OVP, UVP, OCP, thermal protection, thermal propagation, safety & reliability validation  

- 🔌 EV System Integration  
  End-to-end integration of BMS, MCU, Cluster, and Charger systems  

- 📡 CAN Communication & Network Architecture  
  CAN matrix development, signal mapping, diagnostics, and system-level communication  

- 📋 System Requirements Engineering & Documentation  
  OEM audit-grade SRS, RTM (Requirement Traceability Matrix), test case design & verification aligned with standards  

- 🧠 System Development (BMS & MCU)  
  Requirement definition, documentation, validation, and system-level testing  

- 🔍 Root Cause Analysis & Debugging  
  Fault isolation using hardware tools and real-world validation  

- 📏 Standards & Compliance  
  AIS-156, AIS-004, BIS, ISO — validation and certification workflows
